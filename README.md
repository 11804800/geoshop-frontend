
# Geo Shop Next.js Application

A geo-aware e-commerce application built with Next.js that detects user location to display prices in local currency, with options for manual currency selection.


## ðŸ›  Skills
Typescript,Nextjs 16,Tailwindcss,Stripe.


## Features

- Detect user's location and country using IP geolocation
- Automatically display product prices in local currency
- Manually switch currencies between UK (GBP), USA (USD), and India (INR)
- Responsive UI for seamless shopping experience
- Server-side rendering for better performance and SEO



    
## Technologies Used
- Next.js
- IP Geolocation API (e.g., ipgeolocation.io, ipinfo.io)
- Currency conversion API (e.g., ExchangeRate-API, Open Exchange Rates)
- tailwind CSS Modules or Styled Components for styling

## Run Locally

Clone the project

```bash
  https://github.com/11804800/geoshop-frontend.git
```

Go to the project directory

```bash
  cd geoshop-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
